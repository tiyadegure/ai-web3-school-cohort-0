# Handbook Feedback

This directory collects feedback, issues, and suggestions for the [AI × Web3 School Handbook](https://aiweb3.school/zh/handbook/).

## How to Submit Feedback

Each file in this directory should follow this format:

```markdown
# [Short Title]

- **Page**: [URL of the Handbook page]
- **Type**: typo | unclear | outdated | suggestion | missing
- **Date**: YYYY-MM-DD

## Problem

<!-- What's wrong or unclear? -->

## Suggested Fix

<!-- How should it be improved? -->

## Source

<!-- What context led to this feedback? (e.g., "studying Agent Workflow section") -->
```

## Why This Matters

The Handbook is open-source. Your feedback helps improve it for all builders.
Contribute directly: [GitHub](https://github.com/lxdao-official/aiweb3school) | [Telegram](https://t.me/aiweb3school)
