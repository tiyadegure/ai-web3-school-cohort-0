## 提交流程记录

1. **在哪里提交**：WCB Learning 页面（https://web3career.build/zh/programs/AI-Web3-School#tab=learning）或通过 WCB Agent API
2. **Proof 格式**：公开可访问的链接（GitHub repo、X 链接、公开笔记、tx hash 等）
3. **审核者需要看到**：proof 链接指向的内容能证明任务已完成

## Proof 类型说明

- GitHub repo：学习记录、代码实验、概念卡片
- X 链接：公开发布的学习内容
- 公开笔记：Notion / HackMD 等公开页面
- tx hash：测试网交易记录
- 合约地址：部署的测试合约
