# AI × Web3 School — Personal Learning Repo

> Personal learning journal and proof-of-work for [AI × Web3 School](https://aiweb3.school/).

## About

This repo tracks my learning progress in the AI × Web3 School program — an open-source, bilingual learning platform for builders, co-initiated by [LXDAO](https://lxdao.io/) and [ETHPanda](https://ethpanda.org/).

## Key Links

- [Handbook](https://aiweb3.school/zh/handbook/) — The core learning material
- [WCB Course Page](https://web3career.build/zh/programs/AI-Web3-School) — Bootcamp info & schedule
- [WCB Learning Page](https://web3career.build/zh/programs/AI-Web3-School#tab=learning) — Daily tasks & check-in
- [GitHub (upstream)](https://github.com/lxdao-official/aiweb3school) — Handbook source code
