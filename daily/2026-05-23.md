## 今日自学

### 2. Agent Workflow 章节（35 分钟）
Handbook AI × Web3 Bridge 第 3 章，7 个知识节点：

**核心定义：** 把概率模型放进确定性流程里

- **Task Graph** ⭐⭐ — 把目标拆成节点和依赖，每步设置输入/输出/权限/停止条件
- **State Machine** ⭐⭐⭐ — 链上工作流显式状态（draft → confirmed / reverted），系统不会忘记自己在哪
- **Human-in-the-loop** ⭐⭐ — 按风险分层：L1 只读自动 / L3 小额白名单 / L4 高风险必须确认 / L5 越权拒绝
- **Retry/Fallback** ⭐⭐ — Web3 不能盲目重试，交易 pending 不能简单再发一笔
- **Trace** ⭐ — 每步输入/判断/工具调用/结果记录，是复盘的基础
- **Evaluation Harness** ⭐⭐⭐ — 测 Agent 是否正确拒绝越权、识别错误链、在缺数据时停止
- **Regression Set** ⭐⭐ — 固定测试用例防止安全退化（正常swap、错误链、无限approve、恶意文档...）

**详细笔记：** [2026-05-23-agent-workflow.md](2026-05-23-agent-workflow.md)
